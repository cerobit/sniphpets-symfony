# sniphpets-symfony [![Build Status](https://travis-ci.org/sniphpets/sniphpets-symfony.svg?branch=master)](https://travis-ci.org/sniphpets/sniphpets-symfony)

This repository contains [Symfony](http://symfony.com) snippets for the Vim. This snippets were inspired by [ultisnips-php](https://github.com/algotech/ultisnips-php), but this snippets more smart, because they try to guess what you want and have many configuration settings.

## Installation

This plugin requires [Ultisnips](https://github.com/SirVer/ultisnips) and [sniphpets](https://github.com/sniphpets/sniphpets).

### Using [pathogen](https://github.com/tpope/vim-pathogen)

```sh
git clone https://github.com/sniphpets/sniphpets-symfony.git ~/.vim/bundle/sniphpets-symfony
```

### Using [vundle](https://github.com/gmarik/vundle)

Add to your vimrc:

```vim
Plugin 'sniphpets/sniphpets-symfony'
```
